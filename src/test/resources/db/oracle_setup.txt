*) Installing oracle docker on Mac
https://www.petefreitag.com/item/886.cfm

*) Start oracle docker image using docker-compose. File at
$HOME/oracle-install/docker-images/OracleDatabase/SingleInstance/dockerfiles/

docker-compose up

*) get list of docker containers
docker container ls

*) Access the oracle docker container shell
docker exec -it oracle_docker_container_id  bash


*) Now you are on the docker container. On shell prompt type in below. XEPDB1 is pluggable database 'app' is system password setup in docker-compose
sqlplus sys/app@//localhost:1521/XEPDB1 as sysdba

*) On sql prompt type the following:
create user jdbctemplatemapper identified by app;
grant connect, resource, dba to jdbctemplatemapper;
 grant all privileges to jdbctemplatemapper;
